# 每日生產排程看板－依現有表格格式專用版

## 支援的固定欄位
- 製令
- 客戶
- P/N
- Type
- Category
- 組立地點
- 組立人員
- 組立進度
- 備註
- 發料日
- 入庫日
- 客戶入庫日

必要欄位只有：
- 製令
- 客戶入庫日

## 支援的右側日期欄格式
- 06/28
- 07/01
- 2026/06/28
- 2026-06-28
- Excel 原生日期

## 支援的排程代碼
- 組：組立
- T：測試
- Q：品質確認
- R：調整／重工
- B：包裝
- W：等待
- 出：出貨／入庫
- 機：機構
- 配：配線
- 管：管路

公司若有其他代碼，可打開 app.py，在 CODE_LABELS 內新增。

## 使用方式
1. 解壓縮 ZIP。
2. 電腦需先安裝 Python 3.11 以上版本。
3. 第一次可雙擊「安裝套件.bat」。
4. 之後雙擊「一鍵啟動看板.bat」。
5. 瀏覽器會開啟 http://localhost:8501。
6. 上傳原始 Excel 排程表即可。

## 看板功能
- 自動辨識每日日期欄
- 今日作業與未來 7 日作業統計
- 逾期、今日到期、7 日內到期風險
- 客戶、Category、組立地點、風險篩選
- 每日作業負荷圖
- 客戶製令排行
- Category 占比
- 每日排程橫向矩陣
- 異常追蹤清單
- 匯出篩選後 Excel

## 注意
關閉黑色命令視窗後，本機看板會停止。若要不開黑色視窗持續使用，建議部署到 Streamlit Community Cloud 或公司伺服器。
